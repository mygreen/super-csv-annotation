31       * @return {@link CellProcessor}の実装クラスのインスタンス。
32       *         引数nextをそのまま返すため、値がない場合がある。
33       */

35              TextFormatter<?> formatter, Configuration config, BuildCase buildCase, Class<?>[] groups);
