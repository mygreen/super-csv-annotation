プロジェクト情報

この文書は、このプロジェクトの一般的な情報の一部である様々な文書とリンクの概要を示します。すべてのコンテンツはこのプロジェクトのために Maven によって生成されたものです。

概要

文書 説明
このプロジェクトについて CSVのJavaライブラリであるSuperCSVに、アノテーション機能を追加したライブラリです。
Plugin Management This document lists the plugins that are defined through pluginManagement.
Distribution Management This document provides informations on the distribution management of this project.
Dependency Information This document describes how to to include this project as a dependency using various dependency management tools.
ソースリポジトリ ウェブブラウザによって閲覧できるオンラインソースリポジトリへのリンクです。
メーリングリスト この文書はこのプロジェクトのメーリングリストの購読方法と、メーリングリストのアーカイブに関する情報を提供します。
課題追跡 このプロジェクトの課題管理システムへリンクです。このリンクを利用して課題 (バグ、機能、変更要求) を作成し、検索できます。
継続的結合 コードのビルドとテストを定期的かつ頻繁に行う、すべての継続的結合プロセスの定義へのリンクです。
Project Plugins This document lists the build plugins and the report plugins used by this project.
プロジェクトライセンス プロジェクトライセンスの定義へのリンクです。
プロジェクトチーム この文書はプロジェクトのメンバの情報を提供します。これらは、何らかの形でプロジェクトに貢献した人たちです。
プロジェクトサマリ このプロジェクトの関連情報です。
依存関係 この文書はプロジェクトの依存ライブラリをリストアップし、個々の依存ライブラリの情報を提供します。
